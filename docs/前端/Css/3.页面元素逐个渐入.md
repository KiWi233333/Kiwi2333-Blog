# 页面元素逐个渐入

```scss

/* 段落 */
.data-fades > * {
  --lv: 0;
  --delay: 120ms;
  --start: 0ms;
  animation: keywords-enter 1s both 1;
  animation-delay: calc(var(--start) + var(--lv) * var(--delay));
}

@for $i from 1 through 20 {
  .data-fades > *:nth-child(#{$i}) {
    --lv: #{$i};
  }
}

[data-fade] {
  --lv: 0;
  --delay: 120ms;
  --start: 0ms;
}

@media (prefers-reduced-motion: no-preference) {
  [data-fade] {
    animation: keywords-enter 0.6s both;
    animation-delay: calc(var(--lv) * var(--delay) + var(--start));
  }
}

[data-fade-controller="false"] [data-fade] {
  animation: none;
}

```
